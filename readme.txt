== Description ==

This Plugin converts products, product attributes, product variations and more from JigoShop to WooCommerce.

== Installation ==

1. Upload the folder 'woocommerce-jigo-converter' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce - JigoShop Converter' through the 'Plugins' menu in WordPress

== Usage ==

1. Open Tools > Import > JigoShop To WooCommerce Converter

2. Click 'Convert Now' button

That's it!
